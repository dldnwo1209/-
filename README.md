# 🏫 4단계 학급 회계 관리 시스템

이 프로그램은 학생 중심의 학급 운영을 위한 회계 실습 도구입니다.

## 👥 사용자 권한 안내
- **교사**: 부처별 총 예산을 설정하고 전체 재정을 모니터링합니다.
- **총무**: 부장의 품의서를 검토하여 승인하거나 사유와 함께 반려합니다.
- **부장**: 예산 범위 내에서 물품 구매를 신청하고 결과를 확인합니다.
- **감사원**: 모든 결재 이력을 감시하고 예산 소진 현황을 시각화하여 확인합니다.

## 🚀 배포 가이드
1. GitHub 리포지토리에 `app.py`, `requirements.txt`를 올립니다.
2. Streamlit Cloud에 연결하여 배포합니다.
